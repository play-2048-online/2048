## 🕹️ How to Play 2048

1. **Swipe Tiles**: Move all tiles in one direction.
2. **Merge Same Numbers**: Two tiles with the same number combine into one.
3. **Reach 2048**: Create a 2048 tile to win — but you can keep playing!
